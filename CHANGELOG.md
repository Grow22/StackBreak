# 변경사항 (5/20)

## 기능 변경

### 총알 (Gun) 시스템 개선
- **총알 속도 2배**: 틱당 1칸 → 2칸 이동으로 변경
- **보스 명중 판정 수정**: 총알이 천장에 도달해도 보스 위치(`piece_c ~ piece_c+1`)와 겹칠 때만 HP 감소. 빗나간 총알은 데미지 없이 소멸

### 아이템 분배 알고리즘
- 보드를 3구역(좌 0-3 / 중 4-6 / 우 7-9)으로 나누어 아이템 밀집 구역 추적
- 아이템이 몰린 구역에는 55~90% 확률로 **반대편에 생성**되도록 가중치 적용
- 완전 랜덤은 아니므로 유저가 100% 예측 불가

### 점프 물리 개선
- **코요테 타임** (5틱): 발판에서 떨어진 직후에도 잠깐 점프 가능
- **점프 버퍼** (6틱): 착지 직전에 미리 점프키를 눌러도 착지 즉시 점프 실행

### 콤보 시스템
- 줄 클리어 후 2초(60틱) 이내에 추가 클리어 시 콤보 배수 적용
- 콤보 보너스: `콤보 수 × 50 × 레벨`

### 블록 중력
- 줄 클리어 후 빈 공간 아래로 블록이 자동으로 떨어지는 컬럼 중력 적용

### 기타
- Makefile: `ncurses` → `ncursesw` 링크로 변경 (유니코드 지원)
